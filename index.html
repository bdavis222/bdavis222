Projects

PyPI Maker

Developed a desktop application to enable inexperienced users to more easy distribute their Python packages via PyPI (the Python Package Index). The application is entirely GUI-based, and guides the user through the submission process step by step, automating all of the tedium involved in the traditional process. The project can be viewed here. (Python)

Operator Game

Developed an original iOS math puzzle game using the Model-View-ViewModel (MVVM) architectural design pattern in SwiftUI. The puzzles are procedurally generated, and the difficulty adapts to player performance, providing a challenge for all ages. A demo of some of the gameplay can be viewed here. (Swift)

Dot Scanner

Created an industry-leading desktop application (macOS and Windows) for microscopic image analysis. This window-based GUI application features original algorithms to scan noisy microscope images of plant cells to differentiate faint particles from similar unwanted features within the cells and calculate particle densities and lifetimes within user-selected study regions. The project can be viewed here. (Python)

Gravty Game

Developed an original game in which the player uses gravitational fields of planets to navigate their spaceship toward a target planet. The game features a custom physics engine built from the group up with multiple classes to handle animations and in-game events. A demo of some of the gameplay can be viewed here. (C++)

Cluster Member Sorter

Designed a program to identify whether specific star clusters in our Milky Way galaxy host a rare class of star. The program uses Gaussian mixture modeling algorithms and large datasets pulled from astronomical databases. This software enabled the astronomical community’s most comprehensive list of these stars to be compiled. The project can be viewed here. (Python, SQL)
